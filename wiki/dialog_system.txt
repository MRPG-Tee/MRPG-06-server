Dialogs have 3 types of formatting

{} - operations
[] - preinit
<> - runtime

Operations {}
- {OR} (Allows you to customize the dialog with random)

Preinit [] - is set at the beginning of the dialogue
- [p] - player says in dialogue
- [Bot_DataBotID] - bot says in dialogue

Runtime
- <Bot_DataBotID> - replace text to bot name
- <World_WorldID> - replace text to world name
- <Player> - replace text to player name
- <Time> - replace text to server time string
- <Here> - replace text to own world name
- <TalkingTo> - replace text to who the player is talking to
